# Retail Sales Data Analysis using SQL

## Project Overview
This project demonstrates basic SQL data analysis techniques using a retail sales dataset. The objective is to analyze product performance, regional sales distribution, and overall revenue trends.

## Tools Used
- SQL
- Excel (for dataset creation)

## Dataset Description
The dataset contains information about retail sales transactions including:

- Order ID
- Product Name
- Category
- Region
- Sales Amount
- Order Date

## Business Questions
This project answers the following questions:

1. What is the total revenue generated?
2. Which products generate the highest revenue?
3. Which region has the highest sales?
4. Which category performs the best?
5. What are the monthly sales trends?

## Key Insights
- Electronics generated the highest revenue.
- The South region had the largest sales contribution.
- Laptop and Phone were the top-performing products.

## Files in this Repository

sales_data.csv – dataset used for analysis  
sales_analysis.sql – SQL queries used to analyze the data
